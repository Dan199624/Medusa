# Medusa

A mod menu base for Grand Theft Auto V. Strictly for educational purposes.

This base was made relatively fast, so some code may not be the best. 
I will try my best to keep this base up to date.

## Features
+ Modernized ui
+ Fast pattern scanner
+ Uses modern C++
+ Unloading at runtime
+ Simple fiber system

## Projects Used
+ MinHook (github.com/TsudaKageyu/minhook)

## How to Build
```
git clone https://github.com/Dan199624/Medusa.git
```
Build on Release and /std:c++latest
